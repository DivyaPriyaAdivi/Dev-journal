What is Dev Journal?

Dev Journal is a personal developer log app where we can document programming errors, debugging experiences, and solutions encountered during real-world development. Instead of losing valuable learning moments, this journal preserves them as structured knowledge for future reference.

current feature- #1

When we working a particular problem, there is unavoidable cases where we face bugs and we should be glad it occurs because it is what make us learn and understand things better.so while trying to solve it , we tend to browse and look up articles , approach LLMs and reading stack overflow and reddit for solution and what if there will be situation , that you gonna face it again so instead of browsing it again we can clip those articles that helped you solve the bug previously under the post that you created with title that match the bug.it will reduce the time of debugging.for that I made a clipper extension in firefox browser where you can clip the current tab of the webpage and it will save the link in the post you have selected.
